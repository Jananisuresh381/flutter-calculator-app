# Flutter Calculator App

A simple calculator application built using Flutter. This app supports basic arithmetic operations with a clean and minimal user interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear and responsive UI

## Technologies Used
- Flutter
- Dart
      
## Purpose
This project was developed to practice Flutter widgets and basic application logic.
